# 📊 Bank Loan Analytics Dashboard

## Project Overview

The Bank Loan Analytics Dashboard is an interactive Power BI project developed to analyze loan portfolio performance, borrower behavior, loan quality, and lending trends. The dashboard transforms raw loan data into meaningful business insights that help stakeholders make informed lending and risk-management decisions.

---

## 🎯 Project Objectives

- Analyze overall loan portfolio performance.
- Monitor funded and recovered loan amounts.
- Evaluate loan quality through Good Loan and Bad Loan analysis.
- Identify borrower trends across states, employment length, loan purpose, and home ownership.
- Support data-driven decision-making through interactive visualizations.

---

## 🛠️ Tools & Technologies Used

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query
- Data Modeling
- Data Visualization

---

## 📌 Dashboard Pages

### 1️⃣ Introduction
Provides an overview of the project, objectives, and key areas of analysis including portfolio performance, loan quality assessment, borrower insights, and risk evaluation.

### 2️⃣ Executive Summary
Displays key performance indicators (KPIs):

- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average DTI (Debt-to-Income Ratio)
- Good Loan Percentage
- Bad Loan Percentage

Also includes:
- Good vs Bad Loan Analysis
- Loan Status Performance Matrix
- Month-to-Date (MTD) Metrics
- Month-over-Month (MoM) Growth Analysis

### 3️⃣ Overview Analysis
Provides deeper insights into lending patterns through:

- Monthly Loan Application Trends
- State-wise Loan Distribution
- Loan Term Analysis
- Employment Length Analysis
- Loan Purpose Analysis
- Home Ownership Analysis

### 4️⃣ Detailed Loan Records
Offers transaction-level loan details with dynamic filtering capabilities.

Filters Available:
- State
- Grade
- Good vs Bad Loan

---

## 📈 Key Insights

- Over 86% of loans are classified as Good Loans.
- Total loan funding exceeds $435M.
- Total repayments exceed $473M, indicating strong portfolio recovery.
- Debt Consolidation is the most common loan purpose.
- Borrowers with longer employment histories account for a significant share of loan applications.
- Geographic analysis highlights key lending regions and borrower concentrations.
